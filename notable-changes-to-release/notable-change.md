# KEB: Taints Support for Additional Worker Node Pools

> ### Caution:
> This update is mandatory. Without performing it, you will not be able to use the feature in the SAP BTP cockpit.


## Prerequisites

Access to Environment Registry Service (ERS).

## What's Changed

You can set **taints** in additional worker node pools.

## Procedure

Refresh broker details using the xRS APIs in ERS.

## Post-Update Steps

Verify that in the SAP BTP cockpit, the **taints** field is visible in the configuration window under the **Additional Worker Node Pools** section.
