# Updating Kyma Environment Broker: Updatable Cluster Name

> [!WARNING]
> This update is mandatory. Without performing it, you will not be able to update the cluster name.

## Prerequisites

- Access to the ERS registry.

## What's Changed

- The **Cluster Name** is now an updatable parameter.

## Procedure

1. Refresh broker details using the XRS APIs in the ERS registry.

## Post-Update Steps

1. Confirm that the cluster name is visible in the SAP BTP cockpit and can be updated successfully.
